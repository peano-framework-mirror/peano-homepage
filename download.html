Download

Peano is a C++ class collection used by Peano applications. Hence, download the kernel first before you test any application. For Peano developers, i.e. for researchers creating Peano applications from scratch, there is the Peano Development Toolkit (PDT), a small Java tool. It is available here both as jar file and as source code. Besides the kernel and the toolkit with its sources, the Peano webpage also hosts a couple of example applications that are open to the public and development resources.

All files are hosted at sourceforge.net and can be downloaded from this page or due to the deep links below. Alternatively, you may check out the data directly from the repository: 

svn://svn.code.sf.net/p/peano/code/trunk/kernel

The PDT is available at 

svn://svn.code.sf.net/p/peano/code/trunk/pdt

It is based upon the tool DaStGen embedded into the jar file. If you build the PDT yourself, you might have to download DaStGen's jar files separately.
